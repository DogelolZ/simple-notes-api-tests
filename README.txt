SimpleNotes API Tests

GitHub repository URL:
https://github.com/DogelolZ/simple-notes-api-tests

Upload to GitHub:
1. Open the folder named SimpleNotesApiTests.
2. Upload everything inside that folder to the root of your GitHub repository.
3. After upload, the repo root should contain:
   - SimpleNotesApiTests.csproj
   - DTOs folder
   - Tests folder
   - .github folder
   - .gitignore
4. Wait for the workflow in the Actions tab.
5. When the workflow passes, zip the SimpleNotesApiTests folder together with GitHub-Repo-URL.txt and submit.
